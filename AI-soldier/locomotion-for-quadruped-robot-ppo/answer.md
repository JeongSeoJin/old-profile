## 보행 인공지능 모델을 직접 구현하신 건가요?

"핵심 알고리즘은 검증된 라이브러리를 활용했고, 학습 환경(Environment) 설계와 보상 함수(Reward Function) 최적화를 직접 구현했습니다."

- 상세 설명:

"알고리즘은 Stable Baselines3의 PPO를 사용하여 안정성을 확보했습니다."

"하지만 로봇이 실제로 걷게 만드는 것은 보상 설계에 달려 있습니다. 저는 Unitree Go1의 물리적 특성을 고려하여 Gymnasium 기반의 환경을 직접 구축했고, 자연스러운 보행을 유도하기 위해 Reward Engineering과 하이퍼파라미터 튜닝을 수행했습니다."

----딥하게----

"End-to-End 학습 **파이프라인(Pipeline)**을 직접 구축했습니다."

상세 설명:

"MuJoCo 시뮬레이터와 강화학습 에이전트를 연결하는 Gymnasium 인터페이스를 직접 작성했습니다."

"단순히 라이브러리만 돌린 것이 아니라, 로봇의 State(관절 각도/속도)와 Action(토크 제어) Space를 정의하고, Sim-to-Real을 고려한 관측 노이즈 및 보상 함수를 설계하여 보행 정책(Policy)을 학습시켰습니다."
